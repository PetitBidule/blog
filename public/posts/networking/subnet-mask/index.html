Dans un réseau avec un masque de sous-réseau /24 ou (255.255.255.0), il y a exactement 256 adresses IP possibles. Explication: Un masque /24 signifie que les 24 premiers bits sont utilisés pour l’adresse du réseau soit l’équivalent de la class C, et les 8 bits restants sont utilisée pour les hôtes.

  • Une adresse est réservée pour l’identifiant du réseau
  • Une adresse est utilisée comme adresse de diffusion (broadcast) Soit un total de 254 adresses dispo pour les hôtes.

Dans un réseau avec un masque de sous réseau /16 soit la class B, il y a environ 65 000 ip diso Dans un réseau avec un masque de sous réseau /8 encore plus
