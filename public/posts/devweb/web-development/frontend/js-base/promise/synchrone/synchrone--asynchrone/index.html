Synchrone

Du code synchrone est du code qui s’exécute ligne après ligne.

Asynchrone

Le JS a comme particularité d’etre capable de fonctionner de manière asynchrone et peut donc continuer à exécuter du code pendant certaines opérations.

Fonction asynchrone

La async function déclare une fonction asynchrone ou le await mot-clé est autorisé dans le corps de la fonction. Les mots-clés async et await permettent d’écrire un comportement asynchrone basé sur des promesses (voir cours promesse) dans un style plus propre.
